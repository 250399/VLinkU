------------

   http://php.net/docs.php
   for further information.

CONFIGURATION
-------------
The module has no menu or modifiable settings. There is no configuration. When
apache or sql service is stopped, the module will prevent the links from appearing. 
To get the links back, start the service.

Troubleshooting & FAQ
---------------------
If sql service is not starting, check the following:
		-Port is available
		-mysqld is already running in background, make sure you close it using
		task manager
